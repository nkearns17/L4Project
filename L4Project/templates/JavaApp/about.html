Evaluation Aim


The aim of this experiment is to investigate the suitability of this web application for the use of learning and revising Java programming. We cannot tell how good web applications are unless we ask those people who are likely to be using them, which is why I need to run experiments like these to see just how effective using interactive activities are when learning and revising a programming language. I will give you some time to browse the pages and try out the activities, before asking you to fill out a short questionnaire about the usability and effectiveness of this web application. Please feel free to email me or Jeremy Singer if you have any questions or problems.

Once you have browsed the pages and tried out the activities, please could you take a few minutes to complete this questionnaire.

About JavaApp


The aim of this application is to allow users to learn and practice the basic Java concepts and syntax. In order to do this, I have created the following activities:

  • Fill in the blanks
  • Multiple choice to predict the behaviour of the code
  • Write your own code
  • Fix the broken code
  • Video tutorials
  • Tutorials to go over the basics

Through these interactive activities, we hope that students will enhance the their ability to program using Java.
